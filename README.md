# ☕ Khóa học Java Cơ Bản - Anh Tester (01/2026)

Chào mừng bạn đến với repository **Java Basic** thuộc khóa học Automation Testing (Selenium Java) do [Anh Tester](https://anhtester.com/) tổ chức.

Repository này chứa các bài học thực hành về ngôn ngữ lập trình Java ở mức cơ bản, giúp bạn xây dựng nền tảng vững chắc trước khi bước vào tự động hóa kiểm thử (Automation Testing).

## 📚 Nội dung mã nguồn (Source Code)

Các bài học trong khóa học được phân chia theo từng thư mục tương ứng với từng chủ đề cụ thể trong thư mục `src/`:

- **Variables**: Các loại biến và cách khai báo biến trong Java.
- **DataType**: Các kiểu dữ liệu cơ bản và nâng cao thường dùng.
- **ToanTu**: Các toán tử và phép tính cơ bản.
- **DieuKienIFELSE**: Cấu trúc rẽ nhánh điều kiện If - Else.
- **SwitchCase**: Cấu trúc rẽ nhánh Switch - Case.
- **VongLapFOR**: Các loại vòng lặp (For, While, Do-While).
- **MangMotChieu**: Mảng tĩnh một chiều (Array).
- **PhuongThuc**: Cách khởi tạo và sử dụng Phương thức (Method).
- **StaticJava**: Nắm vững từ khóa `static` trong Java.
- **XuLyChuoiString**: Các hàm xử lý chuỗi thường gặp.
- **DataCollections**: Các cấu trúc dữ liệu collection (ArrayList, HashSet, HashMap,...).

## 💻 Cách sử dụng

1. Clone repository này về máy tính của bạn:
   ```bash
   git clone https://github.com/anhtester/JavaBasic012026.git
   ```
2. Mở project bằng IDE yêu thích (IntelliJ IDEA, VS Code,...).
3. Mở các class trong từng thư mục thuộc `src/` và chạy (Run) trực tiếp hàm `main` để xem kết quả qua Console.
